  ## 运行项目

  ## 服务器端返回
  - 输入 url 或者点击一个链接(死板)
  - 向服务器端发送请求
  - node/java 请求 去数据库取数据，生成html字符串
  - 返回Html字符串

  ## http 请求
  - 请求行  
    GET  http://www.baidu.com
    POST http://api.deepseek.com/chat/completions
  - 请求头
    设置各种头部信息
    {
      Content-Type: 'application/json',
      Authorization: 'Bearer xxx',  // 请求凭证
    }
  - 请求体
    发送给服务器的数据
    GET 没有请求体
    POST 有请求体
