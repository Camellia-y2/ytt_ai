## JS 数据类型
- primitive 简单数据类型 栈内存
    拷贝式传值
String Boolean Undefined Null  Symbol  Numeric(bigint number)
    Symbol 是 JavaScript 中的一种简单数据类型，在 ES6 中引入，用于创建唯一且不可变的值，常被用作对象属性的键以避免命名冲突。
- Non-primitive 复杂数据类型 栈内存（连续）（地址的引用） + 堆内存（对象内容具体的值）
    引用式赋值
object

JS 有哪些数据类型？
答： 7种
String Boolean Undefined Null  Symbol  Numeric 6中primitive类型
其中numeric 又分为：bigint number
其他的都是 object 类型
